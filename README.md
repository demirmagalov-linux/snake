# Snake

A Snake game built with Python and Ursina Engine. Eat food to grow longer and avoid hitting the walls or yourself.

## How to play

Run the game with:

python snake.py

Controls:

- Arrow keys: move the snake

Eat as much food as possible to increase your score. The game ends if you hit the walls or collide with your own body.

## Features

- Simple and responsive controls
- Increasing difficulty as the snake grows
- Score tracking
- Game over screen

## Requirements

Python 3 and Ursina Engine.

To install Ursina:

pip install ursina

## Notes

The snake moves continuously in the direction you press. Each time you eat food, your snake grows longer and the game becomes more challenging.
